# Cloud Runner

Pygame Zero ile geliştirilmiş 2D platform oyunu. Karakterinizi kontrol ederek coinleri toplayın, düşmanlardan kaçının ve bayrağa ulaşarak seviyeyi tamamlayın

## Özellikler
- Akıcı animasyonlu karakter ve düşmanlar
- Coin toplama sistemi
- Duraklatma ve ses kontrolü
- Kazanma/kaybetme ekranları

## Kurulum
```bash
pip install pgzero
pgzrun cloud_runner.py
